1️⃣ Core Java Fundamentals

If you can’t do these comfortably, pause and strengthen them:

Creating classes

Constructors

Methods

ArrayList

HashMap (optional but useful)

Loops and conditionals

Basic exception handling (try-catch)

You don’t need advanced Java. Just solid basics.

2️⃣ File & Directory Handling (Most Important)

You must know how to:

Create a directory

Check if a file exists

Create a file

Read file content

Write content to a file

Append to a file

Specifically understand:

File

FileWriter

FileReader

BufferedWriter

BufferedReader

Without this, your project cannot exist.

3️⃣ How Git Works Conceptually

Not advanced Git.

Just understand:

Git works by:

Creating snapshots of files

Storing metadata (message + timestamp)

Assigning IDs to commits

You are not cloning Git.
You are copying its idea.

Understand the idea. Not the tool.

4️⃣ Basic System Design Thinking

You must be able to answer:

What is a repository?

What is a commit?

Where will I store data?

How will I identify versions?

If you can’t explain your own structure on paper, don’t open the IDE.

5️⃣ (Optional but Powerful) Hashing

If you want stronger design:

Learn how to generate SHA-256 hash from a string.

This can become your commit ID.

It teaches:

Byte arrays

Encoding

Deterministic ID generation
